# Pop-up Abençoado

## Overview

Sistema fullstack de notificações motivacionais que executa ao iniciar o sistema, coleta o humor do usuário e exibe mensagens personalizadas.

---

## Stack

- React.js
- Node.js
- Express.js
- Socket.IO
- MySQL

### Frontend

- React.js
- Socket.IO Client

### Backend

- Node.js
- Socket.IO
- Express.js

---

## Features

- Notificação obrigatória (popup)
- Seleção de humor
- Mensagem motivacional dinâmica
- Comunicação em tempo real
- Arquitetura modular

---

## Arquitetura

- Event-driven
- Comunicação via WebSockets
- Separação frontend/backend
